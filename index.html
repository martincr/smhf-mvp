FAQ

Is this more like Opendoor or a lead marketplace?

For the MVP, it should be a lead marketplace. The site gathers seller info, Rush organizes it, and the lead goes to buyers who can make cash offers quickly.

Should the first field be ZIP code or address?

For speed, use “ZIP code or address” and let the seller choose. ZIP is easier for a lightweight MVP, but address-first usually feels more concrete for high-intent sellers.

Should I scrape Google Images of the property?

No. Start with seller uploads. Later, you can add a legitimate address lookup and optional map or street-view style visuals.

What is Rush?

Rush is the short AI agent name in this template. Rush does intake cleanup, lead card creation, and buyer routing. It does not need to pretend to be a full valuation engine on day one.

Can I use SMHF.com?

Yes as a short, memorable domain if you control it, but always pair it with the full words “Sell My House Fast” in the logo, title tags, and headline so the offer is instantly clear.
